KMP Plugin For FCKeditor:

Add Description:

1. Upload "dewplayer" folder to  fckeditor/editor/plugins 

2. Modify Fckeditor/fckconfig.js Content:
    Increase the line of code:
		FCKConfig.Plugins.Add( 'dewplayer','en,zh-cn') ;  //DewPlayer Mp3 Plugin
    
3. Add a value 'DewPlayer' to FCKConfig.ToolbarSets
